STCaptureSessionUserInstruction

Indicates a set of warnings that a user of the capture session should act upon.

All values can be combined together into a single integer that can express an array of options. To separate these options, one can use the bitwise-AND operator (&) to check for each warning.

bool needToConnectSensor = _captureSession.userInstructions & STCaptureSessionUserInstructionNeedToConnectSensor;
