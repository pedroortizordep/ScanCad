STOccFileWriter Class Reference

Inherits from NSObject
Declared in STOccFileWriter.h

Overview

The OCC writer class is the central class that manages writing incoming sensor data from a capture session to an OCC file.

This class will be automatically created and managed by the STCaptureSession, however the following interface is exposed when calling captureSession.occWriter.

– startWriting:appendDateAndExtension:

Starts writing events to the OCC file at the specified file path property.

- (BOOL)startWriting:(NSString *)occFilePathToWriteOrNil appendDateAndExtension:(BOOL)appendDateAndExtension

Discussion

If the file path is set to nil or the empty string, writes the OCC to a default file path in the app’s AppDocuments directory following the format:

[AppDocuments]/STSession_yyyy-MM-dd_hh-mm-ss.occ
